# 🧘 jieya-lab · 解压方式科学评级器

你下班后瘫在床上刷两小时手机，以为是在放松。
其实你在**给焦虑充值**。

这个工具把 20+ 种常见解压方式按「证据」重新排了个座次，告诉你：
你今天的解压，到底是在**还债**，还是在**借新债**。

## 它凭什么打分

它把散落在各处的研究结论，折算成一个可比较的「每分钟解压效率」：

- **音乐疗愈**：多项 RCT 荟萃分析显示能显著降焦虑（效应量 SMD 约 −0.4 到 −0.8），
  机制是促多巴胺、压皮质醇——**效率最高的一档**。
- **刷手机 / 刷负面新闻（doomscrolling）**：研究反复证实它抬高皮质醇、
  让神经系统持续戒备，**越刷越焦虑**——所以它的分数是**负的**，是「负资产」。
- **捏捏乐**：能带来即时触觉安抚，但 2025 年抽检 11 款样品有害物质超标率 **100%**，
  工具给正分的同时打上安全警示。

## 它输出什么

- **净解压分**：你今天的解压是正是负，一眼看清。
- **每分钟效率排行榜**：同样是「放松」，含金量差了几十倍。
- **负资产清单**：哪几项是当下爽、事后更焦虑的「借债」。
- **换算建议**：同样的时间换个活法，能多赚多少分。

## 用法

```bash
python3 jieya.py --do "刷手机:60,听歌:20,撸猫:15"
python3 jieya.py --do "喝酒:40,刷短视频:90,熬夜:60" --json
python3 jieya.py --list          # 看完整证据库
```

零依赖，纯标准库，不联网。支持随口写：刷抖音、散步、吸猫、泡澡…都能识别。

## 一个扎心的结算

```
刷手机:60,听歌:20,撸猫:15
→ 净解压分：-6   🔴 今天你在给焦虑充值

  听歌   20分 × +1.8 = +36
  撸猫   15分 × +2.0 = +30
💸 刷手机 60分 × -1.2 = -72   ← 一个人抵消掉所有

🔁 把 60 分钟「刷手机」→「快走」，净赚 +228 分
```

**最贵的解压，是刷完手机的那两小时——它连账单都不给你看。**
