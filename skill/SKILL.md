# 解压方式科学评级器

核心论点：**解压不是「做点什么让自己爽」，是「别在放松时偷偷借债」。
最贵的解压是刷完手机那两小时——当下爽，事后皮质醇更高、更空虚。**

证据库把研究结论映射成可加总的「每分钟解压效率」：音乐疗愈（RCT 荟萃 SMD
约 −0.4~−0.8 降焦虑）、运动、亲近自然、撸宠物是高分正资产；
刷手机/刷短视频/刷负面新闻（doomscrolling 抬皮质醇）、喝酒、报复性进食/熬夜
是负资产。捏捏乐给正分但附 2025 抽检有害物超标率 100% 的安全警示。

## 用法

```bash
python3 <skill-dir>/jieya.py --do "刷手机:60,听歌:20,撸猫:15"
python3 <skill-dir>/jieya.py --do "喝酒:40,刷短视频:90" --json
python3 <skill-dir>/jieya.py --list    # 完整证据库
```

零依赖纯标准库，不联网。别名丰富：刷抖音/散步/吸猫/泡澡都能识别。

## 回答用户时的要点

1. **先跑再讲**：把用户描述的今日/日常解压折成 `--do "方式:分钟,..."` 跑一遍，
   用「净解压分是正是负」和「负资产扣了多少分」开场。负分最有冲击力。
2. **核心反直觉点**：很多人以为「刷手机=放松」，工具的价值就是把这个负资产
   显式算出来，并给等时长的正资产替换（刷手机→快走/听歌）。这是全工具的钩子。
3. **别道德绑架**：负资产不是"不许做"，是"知道它在借债"。偶尔刷手机、喝一杯
   没问题，工具是帮人看清结构、做小替换，不是让人愧疚。
4. **口径诚实**：效率分是文献效应量方向与强度的可比映射，不是精确疗效值；
   个体差异大。心理长期困扰（持续失眠/抑郁情绪）建议专业心理咨询或精神科，
   别用解压小工具替代。
5. 用户问「那到底该干嘛」：给证据最强的三档——运动/快走、音乐疗愈、
   亲近自然或倾诉，按他的场景挑一个能立刻做的。
